  • React Music Library

    React, Firebase, Netlify, Bulma, Spotify API

    This project was started so that I could learn React through doing, and to get to grips with a few other bits of tech aswell. The idea came about when each time I went to a music store to buy new CDs, I would always forget what I did and didn't have, so having a library I could reference would be useful to keep track of my collection.

    I decided to use Bulma for the styling as I'd become familiar using it in other personal projects, and it allowed me to focus purely on the functional side of the app. I then setup a series of cloud functions in Netlify which would allow me to interact both with Spotify to collect an API Key & artist/album data and Google's Firebase, where I would store the saved artist/album IDs.

    The functionality of the publicly accessible one has been locked so as to prevent messing about, but eventually I aim to create a user-specific version, where anyone can have an account.
